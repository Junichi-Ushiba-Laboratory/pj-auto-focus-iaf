## pj-auto-focus-iaf
### auto-focus IAF
1. fcn_removeF eliminate 1/f aperioic component from the power spectral density.
2. The maxima of residual will be set as IAF
3. Find IAF iteratively
4. Use 1-D gaussian fitting with acuquired IAFs
